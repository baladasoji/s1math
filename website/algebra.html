Algebra: Formulae, Tables, and Graphs

S1 Algebra focuses on spotting and describing patterns, building rules for sequences, and using those rules to make predictions. You will also learn to simplify algebraic expressions and solve simple linear equations.

Pictorial Sequences

What is a pictorial sequence?

A pictorial sequence is a pattern made from shapes — such as tiles or matchsticks — that grows in a regular way from one step to the next. Each step adds the same number of shapes as the previous step.

Example — a growing tile pattern:

Each step adds one extra column of tiles.

Step 1       Step 2       Step 3
[ ]          [ ][ ]       [ ][ ][ ]
             [ ][ ]       [ ][ ][ ]

Step 1 has 2 tiles, Step 2 has 4 tiles, Step 3 has 6 tiles. The number of tiles increases by 2 each time.

Why pictorial sequences matter

Looking at a growing picture helps you see why the numbers change the way they do. The picture makes the rule visible before you write it in symbols.

Arithmetic Sequences

Definition

An arithmetic sequence is a list of numbers where each term increases (or decreases) by the same fixed amount, called the common difference.

Example:

3, 7, 11, 15, 19, …

Common difference = 4 (we add 4 each time).

This matches the tile pattern above: 2, 4, 6, 8, … has common difference = 2.

Finding the next term

To find the next term, simply add the common difference to the current term.

Example:

Sequence: 3, 7, 11, 15, …

Next term = 15 + 4 = 19

Term after that = 19 + 4 = 23

General Rule (nth Term)

What is the nth term?

Instead of listing every term one by one, we can write a general rule — also written T(n) — that gives the value of any term directly from its position number n.

How to find the nth term of an arithmetic sequence
  • The common difference d tells you the multiplier of n.
  • Work out d × 1 and compare with the first term to find the constant to add or subtract.
  • Write the rule as T(n) = d × n + c.
Example:

Sequence: 3, 7, 11, 15, … (common difference = 4)

Multiply n by 4: 4, 8, 12, 16, …

Compare with sequence: each value is 1 less than 4n, so c = −1.

T(n) = 4n − 1

Check: T(1) = 4(1) − 1 = 3 ✓   T(2) = 4(2) − 1 = 7 ✓

Using the rule to find any term

Substitute the position number into the rule. This lets you jump straight to any term without listing every one in between.

Example — using T(n) = 4n − 1:

10th term: T(10) = 4(10) − 1 = 40 − 1 = 39

100th term: T(100) = 4(100) − 1 = 400 − 1 = 399

Table of Values

Organising a sequence in a table

A table of values pairs each term number n with its term value T(n). This makes the pattern easy to read and to plot.

Example — sequence 3, 7, 11, 15, 19:
n (term number) 1 2 3 4 5
T(n) (term value) 3 7 11 15 19

Plotting a Sequence

How to plot the sequence

Use the table of values to plot the points (n, T(n)) on a coordinate grid — n on the horizontal axis and T(n) on the vertical axis.

Points to plot from the example above:

(1, 3)   (2, 7)   (3, 11)   (4, 15)   (5, 19)

T(n)
20 |                         *
16 |                   *
12 |             *
 8 |       *
 4 | *
 0 +----+----+----+----+----+-- n
   0    1    2    3    4    5
Key observation

The points of an arithmetic sequence always lie in a straight line. This is because the term value increases by the same amount (the common difference) for each step along the n-axis.

Simplifying Expressions

Collecting like terms

Like terms contain exactly the same letter(s). You can add or subtract like terms to simplify an expression.

Single variable

a + a + a = 3a

2x + 5x = 7x

10b − 4b = 6b

Mixed terms

3x + 2y + x = 4x + 2y

5a + 3b + 2a + b = 7a + 4b

4m − m + 6 = 3m + 6

Important: You cannot combine unlike terms. 3x + 2y cannot be simplified further because x and y are different letters representing different quantities.

Solving Linear Equations

What is a linear equation?

A linear equation contains a variable raised only to the power 1. Solving it means finding the value of the variable that makes both sides equal. Always do the same operation to both sides to keep the equation balanced.

Type 1 — form: ax + b = c
Example: Solve 2x + 3 = 11

Subtract 3 from both sides:   2x = 8

Divide both sides by 2:   x = 4

Check: 2(4) + 3 = 8 + 3 = 11 ✓

Type 2 — form: ax + b = cx + d
Example: Solve 3x + 1 = x + 9

Subtract x from both sides:   2x + 1 = 9

Subtract 1 from both sides:   2x = 8

Divide both sides by 2:   x = 4

Check: 3(4) + 1 = 13 and 4 + 9 = 13 ✓

Another example: Solve 5x − 2 = 2x + 10

Subtract 2x from both sides:   3x − 2 = 10

Add 2 to both sides:   3x = 12

Divide both sides by 3:   x = 4

Check: 5(4) − 2 = 18 and 2(4) + 10 = 18 ✓

Real-World Connections

Seating arrangements

Tables are pushed together in a row. One table seats 4 people; each extra table adds 2 more seats. The number of seats forms an arithmetic sequence: 4, 6, 8, 10, …   The general rule T(n) = 2n + 2 lets you plan seating for any number of tables.

Saving money

You start with €5 and save €3 every week. Your total savings follow the sequence 5, 8, 11, 14, … with common difference 3. The nth-term rule tells you exactly how much you will have saved after any number of weeks.

Stacking objects

Tins are stacked in triangular layers: 1 tin on top, then 2, then 3, and so on. Counting the total number of tins at each stage produces a sequence that can be described with a general rule.

Fair sharing problems

If 3 friends each chip in the same amount x and together they need €12, then 3x = 12, so x = €4. Simple linear equations arise naturally whenever equal shares or fixed totals are involved.

Key Skills to Develop

  • Recognise and continue a pictorial sequence
  • Find the common difference of an arithmetic sequence
  • Write a general rule T(n) for an arithmetic sequence
  • Substitute a value of n into a general rule to find any term
  • Construct a table of values from a sequence rule
  • Plot (n, T(n)) pairs on a coordinate grid
  • Simplify algebraic expressions by collecting like terms
  • Solve linear equations of the form ax + b = cx + d
  • Check solutions by substituting back into the original equation
