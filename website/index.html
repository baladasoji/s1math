📚 About This Course

Secondary 1 Mathematics is designed to build a solid foundation in mathematical concepts essential for higher-level studies. This curriculum covers four main topics:

  • Numbers: Understanding number systems and their properties
  • Algebra: Symbolic representation and manipulation of mathematical ideas
  • Geometry: Spatial reasoning and properties of shapes
  • Set Theory: Logical structure and mathematical relationships

🎯 Learning Approach

Each topic is organized with:

  • Clear learning objectives and competences
  • Key concepts and definitions
  • Contextual applications and real-world examples
  • Assessment criteria and rubrics
  • Interactive activities and exercises
