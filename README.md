# S1 Mathematics Curriculum - Complete Website

## Project Overview

This project contains a comprehensive, self-contained static website for Secondary 1 (S1) Mathematics based on the European Schools Mathematics Curriculum. The website is designed to help secondary mathematics students learn and apply mathematical concepts in an innovative and accessible way.

## What We Built

### 1. **Complete Static Website**
A fully functional HTML/CSS website with **8 interconnected pages** that can be opened directly in any web browser without requiring a server or internet connection.

### 2. **Curriculum-Based Content**
All content is aligned with the **European Schools Mathematics Syllabus** and covers essential mathematical concepts for Year 1 secondary students.

### 3. **Professional Design**
- Responsive grid layouts that work on desktop and mobile devices
- Color-coded topics for easy visual navigation
- Consistent styling across all pages
- Sticky navigation menu with automatic page highlighting
- Gradient backgrounds and modern UI design

## Website Structure

### Pages and Content

#### **1. Dashboard (index.html)**
- Welcome page with course overview
- Quick links to all four major topics
- Information about the learning approach
- Navigation tips for users

#### **2. Topic Pages**

**Numbers (numbers.html)** 🔢
- Natural, whole, and integer numbers
- Fractions, decimals, and rational numbers
- Irrational and real numbers
- Ratios and proportions
- Number properties (commutative, associative, distributive)
- Divisibility, primes, and factorization
- Powers and roots
- Real-world applications in finance, measurement, and science

**Algebra (algebra.html)** 📐
- Variables, terms, and expressions
- Algebraic operations and simplification
- Linear and quadratic equations
- Systems of equations and inequalities
- Functions and function notation
- Polynomials and special products
- Real-world applications in business, physics, and engineering

**Geometry (geometry.html)** 🎨
- Basic geometric elements (points, lines, planes, angles)
- 2D shapes and polygons
- 3D solids and their properties
- Measurement: perimeter, area, surface area, and volume
- Coordinate geometry and distance formulas
- Transformations (translation, rotation, reflection, enlargement)
- Congruence and similarity
- Real-world applications in architecture, navigation, and design

**Set Theory (set-theory.html)** ⚡
- Set notation and representations (roster, set-builder, interval)
- Set operations (union, intersection, complement, difference)
- Relations and functions
- Domain and range
- Logical operators and reasoning
- De Morgan's Laws and proof methods
- Real-world applications in databases, logic, and probability

#### **3. Learning Objectives (learning-objectives.html)**
Detailed competences organized by topic, including:
- Bloom's taxonomy levels (Remember, Understand, Apply, Analyze, Evaluate)
- Foundation, Application, and Advanced level skills
- Specific sub-objectives for each competence
- Core mathematical competences (problem-solving, communication, reasoning, connections, representation, persistence)

#### **4. Assessment (assessment.html)**
Comprehensive assessment framework including:
- **Formative assessments**: Class participation, exit tickets, homework, quizzes, problem-solving tasks, peer review
- **Summative assessments**: Unit tests, projects, performance tasks, end-of-year examination
- **Grading scale**: 5-point system (Advanced, Proficient, Developing, Beginning, Incomplete)
- **Holistic rubrics**: Detailed criteria for concept understanding, problem-solving, communication, procedural fluency, reasoning, and connections
- **Topic-specific criteria**: Assessment expectations for each mathematical topic
- **Weighting**: 40% unit tests, 20% formative work, 15% projects, 25% final exam

#### **5. Resources (resources.html)**
Teaching materials and learning activities organized by topic:
- Practice worksheets with time estimates
- Classroom activities and games
- Supplementary materials and reference guides
- Tools for teachers and parents
- Usage tips and guidelines

## How to Use the Website

### Opening the Website
1. Navigate to the project folder: `/Users/bbdasoji/Documents/Claude/Projects/European School Math/`
2. Open any `.html` file with your web browser (double-click or right-click → Open With)

4. The website works completely offline - no internet connection needed

- **Topic Cards**: Quick links to explore each mathematical topic
